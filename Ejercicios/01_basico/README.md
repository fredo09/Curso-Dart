## Conceptos basicos del lenguaje Dart :nerd_face:

---

    Todo script hecho en dart, para poder ser ejecutado simpre debe de tener una funcion main.
    Ejemplo:

---

```dart
    main () {
        print ('Hola mundo.');
    }

```
